# Worky Dashboard

Job search pipeline tracker hosted on GitHub Pages.

**Live URL:** https://noabakergillis.github.io/worky-dashboard/

## Features

- ✅ **Persistent data** - Never gets wiped (stored in GitHub, not browser)
- ✅ **Real-time stats** - Queue, sent, replied, total
- ✅ **Filtering** - By pipeline (startup/enterprise) or status
- ✅ **Clean design** - Professional, mobile-friendly
- ✅ **Always accessible** - Works anywhere, any device

## How to Update

Edit `index.html` and push to GitHub. Changes are live within 1-2 minutes.
